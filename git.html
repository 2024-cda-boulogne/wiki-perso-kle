Les commandes Git

Version Git
git version

Git init
git init

Git clone
git clone URL

Git add
git add nom_de_fichier (cela ajoutera un seul fichier)
git add * ( cette option ajoutera tous les fichiers modifiés et nouveaux)

Git commit
git commit -m "message du commit"

Git status
git status

Git branch
git branch
